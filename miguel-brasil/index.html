Meus projetos

STUDYA+

É uma plataforma de software de estudos que apresenta uma nova maneira de como os usuários abordam seus objetivos educacionais. usabilidade mais divertida e imersiva.

Desenvolvidada com Java integrado ao MySQL, e a interface em Java Swing.

LUMISPECT

É uma plataforma gamificada de autoavaliação cognitiva que transforma métodos tradicionais de rastreio de atenção e memória em uma experiência interativa e lúdica através de um questionário interativo.

RESERVAS NSPS

HALLEL

É um site para a comunidade católica Hallel é uma plataforma digital multifuncional, desenvolvida para servir como um centro de informações, comunicação e interação para membros e visitantes. Além disso, conta com uma interface administrativa robusta para gerenciamento eficiente de conteúdo.

Desenvolvida com React Js e Java Spring boot.
